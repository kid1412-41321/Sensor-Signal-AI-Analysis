# 智能传感器信号的深度学习分析与特征提取

## 项目简介
本项目旨在探索利用深度学习技术（如一维卷积神经网络 1D-CNN）对复杂传感器信号进行降噪与分类。这是结合物理器件底层机制与前沿 AI 算法的初步探索。

## 核心特性
* 构建了完整的时间序列信号预处理流水线。
* 实现了轻量级的神经网络架构，探索未来在边缘计算设备部署的可能性。
* 包含完整的数据可视化与模型评估模块。

## 快速开始
1. 克隆仓库：`git clone https://github.com/你的用户名/Sensor-Signal-AI-Analysis.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行训练：`python src/train.py`
